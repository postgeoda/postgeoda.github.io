PostGeoDa is a PostgreSQL extension for spatial data analysis. It aims to provide spatial data analysis functionalities that can be run in SQL:

-- create a Queen contiguity weights: SELECT geoda_weights_queen(gid, the_geom) FROM natregimes; -- compute local Moran: SELECT geoda_local_moran("hr60", queen_w) OVER () FROM natregimes;

PostGeoDa is developed using C and C++ based on the performance oriented libgeoda library. By utilizing spatial index and the architecture of PostgreSQL and PostGIS, it can achieve even better performance than its sibling software GeoDa and libgeoda, and has the ability to handle real big spatial data.
